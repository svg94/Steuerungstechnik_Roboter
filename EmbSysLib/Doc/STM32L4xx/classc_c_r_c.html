Detailed Description

16 bit cyclic redundancy check (CRC) calculation.

Calculates CRC16 with defaults GP = 0xA001 Start = 0xFFFF

Test condition: Data = 0x01,0x23,0x45,0x67,0x89,0xAB CRC = 0xF146
