Detailed Description

Abstract class supporting non volatile memory.

This base class encapsulates non volatile memories, e.g. EEPROM, Flash-RAM or battery buffered RAM

Derived classes should at least:

  • initialize the memory hardware access (but not the memory content)
  • implement writeByte() to write a data byte into the memory
  • implement readByte() to read a data byte from the memory
