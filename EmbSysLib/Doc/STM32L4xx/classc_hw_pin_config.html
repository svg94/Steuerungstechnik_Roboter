Detailed Description

Port pin configuration.

This class provides methods to

  • configure GPIO mode, type, pull-up/-down, speed and alternate function
  • connect a GPIO to an alternate function

bool cHwPinConfig::set ( Function  func,
DWORD  mode = 0 
)
static

Set GPIO to alternate function

A GPIO can be connected to a onchip peripheral module. The enum 'Function' lists all peripheral signals which can be connected to a GPIO. This method uses a application specific mapping 'table'. The application has to define this 'table' on global scope. The entries of this table define on which port pin a peripheral signal is mapped.

Parameters
funcSelected peripheral signal as alternate function
modeSelected pin mode. Use constants defined in MODE, the constants can be combined using bit-or operations.
Returns
  • true, if func is included in table
  • false otherwise
