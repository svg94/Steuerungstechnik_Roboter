char * cDevTextIO::getString ( void  )

Read and interprete input data

Each call to this function will initiate a reading from the hardware as far as new data is available. ESC sequences or function key codes are converted:

Key Code Output
up 0x41 0xC1
down 0x42 0xC2
right 0x43 0xC3
left 0x44 0xC4
Returns
Pointer to last received string or NULL
