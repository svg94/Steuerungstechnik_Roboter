Detailed Description

Periodically call of the update() method of all tasks, which are added to the update list.

There are two ways, how the task handler may be called:

  • Timer interrupt (timer task)
  • Execution of the run() method, in main loop for example (idle task)

To support the cTaskTimer class in a idle task (without timer), another cTaskTimer (with timer) is required.
