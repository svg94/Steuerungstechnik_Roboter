Detailed Description

ISC transport base class.

This class is the base class for ISC transport objects. Derived classes use a communication port to transport ISC data objects. A 16-bit CRC is used.

To create an ISC data object:

  • Define an application specific class with POD subclass named 'Data'
  • This class may have an update() method, which is called on data receive
  • Instantiate an object from the template class cISC::Data. The first template parameter is the name of the aplication specific data class. The second parameter must be an unique ID to identify data objects
