Detailed Description

Abstract class supporting a real time clock.

This base class is an interface to a RTC.

Remarks
When setting or getting, date and time are not check
Depending on hardware, the value of year is maybe limited

Derived classes should at least:

  • initialize the hardware inteface to the RTC
  • implement the methods set() and get()
