Detailed Description

Abstract class supporting the I2C hardware in slave mode.

I2C is a serial communication interface in master/slave mode. This base class handles data transmit/receive actions. The application will become an I2C device
Derived classes should at least:

  • initialize the I2C hardware
  • implement an isr() and call them in the appropriate ISR

The application has to derive cHwI2CslaveDataHandler. The instance of the derived class will then respond to the I2C data transfer interrupt
