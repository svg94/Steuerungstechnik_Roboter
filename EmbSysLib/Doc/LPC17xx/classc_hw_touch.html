Detailed Description

Abstract class supporting touch sensor of graphic displays.

The display position is given by the coordinates (x,y), with a starting point in the (left,top)-corner

Derived classes should at least:

  • initialize the hardware inteface
  • implement the method update()
  • set the member variables xpos, ypos, isTouchedFlag
