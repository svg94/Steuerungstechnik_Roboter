BYTE cTimer::timeout ( WORD  timeToWait_msec,
WORD  timeToWait_usec = 0 
)

Check timeout
The behaviour depends on the run state of the timer:

  • Timer is running: If an timeout occures, the timer will be stopped. All parameters are ignored
  • Timer is NOT running: The timer will be restarted with given parameters
    Parameters
    timeToWait_msecTimer-Laufzeit (ms)
    timeToWait_usecTimer-Laufzeit (us)

    timer state:

    • false: timer is still running
