Detailed Description

Abstract class supporting the SPI hardware in master mode.

SPI is a serial communication interface. This base class handles data transmit/receive actions

Remarks
Interrupts are disabled during data transfer

Derived classes should at least:

  • initialize the SPI hardware
  • implement start() and stop() to control transmission
  • implement transceiveByte() to read/write a data byte from/to the hardware
