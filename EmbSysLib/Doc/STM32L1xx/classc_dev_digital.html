Detailed Description

Access to a digital GPIO port pin.

This class is an interface to digital I/O devices.

This class provides some usefull methods to read from or write to an digital I/O port pin.

Input and output of all methods are defined by a logical view (true or false). The corresponding physical level (high or low) can be defined by constructor.
