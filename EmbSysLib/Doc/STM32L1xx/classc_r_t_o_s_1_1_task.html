Detailed Description

Abstract base class of RTOS tasks.

A RTOS task is a class derived from Task. The derived class has to implement the update() method, which is called by RTOS. Additionaly the onStart() and onStop() methods can be overwritten. This methods are called by RTOS when a task is started or stopped.
